# Lion Roar IL 🦁 — ניתוח התראות מבצע שאגת הארי

ניתוח סטטיסטי של התראות **צבע אדום** במהלך מבצע שאגת הארי — מבצע צבאי משולב של ישראל וארצות הברית נגד איראן, שהחל ב-28 בפברואר 2026.

## תכונות
- ניתוח התראות לפי שעות (0–23) עם הסתברויות
- סינון לפי מחוז / יישוב
- **תכנון נסיעה בטוחה** — השוואת נתיבים חלופיים לפי רמת סיכון
- מפה אינטראקטיבית (Leaflet + OpenStreetMap)
- Slider שעת יציאה עם חישוב סיכון בזמן אמת
- ייצוא לאקסל
- מותאם לנייד (Responsive)

## מקורות מידע
- נתוני צבע אדום: [tzevaadom.co.il](https://www.tzevaadom.co.il)
- מסלולים: [OSRM](https://router.project-osrm.org) + [OpenStreetMap](https://www.openstreetmap.org)
- Geocoding: [Photon (Komoot)](https://photon.komoot.io)

## הפעלה מקומית

```bash
npm install
npm run dev
```

פתח את http://localhost:3000

## טכנולוגיות
- **Next.js 16** (App Router) + TypeScript
- **Recharts** לגרפים
- **Leaflet** למפות
- **XLSX** לייצוא
